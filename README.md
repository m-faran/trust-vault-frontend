1. Install a Algorand Wallet like for eg Pera Wallet
- - [Pera Wallet](https://perawallet.app).
 
2. Turn on Developer settings (be on the testnet) in the Wallet
3. Have some testnet funds
- - [Algorand Testnet Funds!](https://bank.testnet.algorand.network/)
